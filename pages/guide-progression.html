Progression recommandée

Where Winds Meet permet une grande liberté. Tu peux suivre l'histoire principale, explorer ou faire des activités secondaires. Cependant, pour progresser plus facilement, voici une route recommandée.

Début du jeu

1. Faire le Prologue
2. Terminer "Another New Wing"
3. Débloquer la carte
4. Activer les Boundary Stones
5. Apprendre l'arc et Tai Chi

Qinghe

Priorité :

• Histoire principale
• Rencontres
• Exploration
• Jianghu Legacies
• Boss campagne

Kaifeng

Après Qinghe :

• Continuer l'histoire
• Débloquer les nouvelles zones
• Faire les activités secondaires
• Monter l'exploration

Hexi

Quand ton personnage est plus avancé :

• Continuer l'histoire
• Explorer les régions
• Faire les boss importants

À faire régulièrement

  • Activer les Boundary Stones
  • Ramasser les Oddities
  • Faire les défis Meow Meow
  • Ouvrir les coffres
  • Faire les quêtes secondaires

Conseils

• Ne fonce pas uniquement sur l'histoire
• Explore entre les chapitres
• Garde tes ressources rares
• Les boss mondiaux peuvent attendre
